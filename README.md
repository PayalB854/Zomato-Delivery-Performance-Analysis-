Project Name: Food Delivery Performance Analysis (Zomato Case Study)
# 🍕 Zomato Operations Analysis
## 📌 Project Overview
A study on how delivery time impacts customer satisfaction and which cuisines drive maximum revenue.
## 🛠️ Tools
- Python (Data Visualization)
- Business Intelligence Concepts
## 💡 Key Insights
- Customer ratings drop by 0.5 stars if delivery exceeds 40 minutes.
- Weekend revenue is 25% higher, suggesting better fleet management on Saturdays.
